

# Sales Data Dashboard (Excel)

## Project Overview

This project features an interactive **Sales Data Dashboard** built in Microsoft Excel. It is designed to provide comprehensive insights into sales performance, profitability, and customer behavior across different regions and product categories.

## 📊 Dashboard Preview

## 🔑 Key Metrics (KPIs)

* **Total Sales:** 195,217.8
* **Total Orders:** 200
* **Highest Sale:** 1,923.66
* **Total Profit:** 68,287.01

## 📈 Key Insights

* **Top Performance:** Total revenue reached **195K+** from 200 orders, yielding a **$68K profit**.
* **Regional Leader:** The **West** region contributes the highest share of total sales.
* **Category Performance:** **Clothing** and **Books** are the dominant categories, generating the largest portion of sales revenue.
* **Seasonality:** Sales trends show distinct peaks during the months of **April** and **October**.

## 🛠️ Features

* **Interactive Slicers:** Filter data dynamically by **Region**, **Product Category**, and **Customer Name**.
* **Visual Analytics:** * *Bar Chart:* Regional sales comparison.
* *Pie Chart:* Distribution of sales by product category.
* *Line Chart:* Monthly sales trend analysis.


* **Automated Summaries:** A "Key Insights" panel that translates raw data into actionable business findings.

## 📂 Project Structure

* `Sales_Data_Dashboard.xlsx`: The main Excel file containing the dataset, pivot tables, and the visual dashboard.
* `Screenshot 2026-03-13 004100.png`: Image of the dashboard for documentation.

## 🚀 How to Use

1. Clone this repository or download the `.xlsx` file.
2. Open the file in Microsoft Excel.
3. Use the **Slicers** on the right side of the dashboard to filter the data.
4. The charts and KPI cards will update automatically based on your selection.

## 🛠️ Tools Used

* **Microsoft Excel:** Data Cleaning, Pivot Tables, Pivot Charts, Slicers, and Dashboard Design.
